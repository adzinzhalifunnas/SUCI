FAQ

1. Apa Itu SUCI?

Web ini dibuat menggunakan SUCI. Apa itu SUCI? SUCI adalah project open source yang bisa kamu cek di https://github.com/salismazaya/SUCI. Dibuat untuk memudahkan para content creator untuk mengumpulkan donasi/charity. Tidak hanya untuk charity kamu juga bisa menggunakan SUCI untuk keuntungan pribadi ^_^.

2. Kenapa Harus Menggunakan SUCI?

SUCI dibuat bukan untuk keuntungan author. Jadi author membuatnya sebisa mungkin "tanpa pajak". Dengan menggunakan SUCI dana yang dikirim donatur akan langsung masuk ke payment gateway kamu tanpa campur tangan author ^_^.

3. Untuk Siapa SUCI Dibuat?

Tidak hanya untuk streamer/content creator SUCI bisa dipakai siapa saja ^_^.
